# Git

## Install

* Homebrew
* [https://git-scm.com/download/mac](https://git-scm.com/download/mac)

```
brew install git
```

```
git --version
```


## Name and E-mail

```
git config --global user.name 'test'
git config --global user.email 'test@example.com
```


## Color setting

``` git config --global color.ui auto ```
